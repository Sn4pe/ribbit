# Ribbit

Deterministic generative marks from any seed.

## Develop

```sh
bun install
bun run dev
```

## Commands

```sh
bun run check        # lint and formatting
bun run check:types  # Astro and TypeScript diagnostics
bun run format       # apply safe formatting fixes
bun run build        # static production build
```

## Stack

- Astro 7 with static output and strict TypeScript
- Tailwind CSS v4 and Geist Variable
- Interactive theme and responsive navigation implemented in TypeScript
- Landing-page structure
- XML sitemap
- MDX content support
- Svelte 5 for selectively hydrated islands
- Biome and GitHub Actions CI
